# Kestra

This folder contains workflow orchestration exercises using Kestra.

The exercises were developed as part of the Data Engineering Zoomcamp orchestration module.

## Goal

The goal of this section is to understand how Kestra can be used to orchestrate data pipelines, manage task dependencies, run scheduled workflows and integrate with Google Cloud services.

## Topics covered

- Kestra flows
- Tasks and dependencies
- Docker-based task execution
- Python scripts inside Kestra flows
- ETL and ELT concepts
- Google Cloud Storage integration
- BigQuery dataset and table creation
- KV Store usage
- Secrets usage
- Application Default Credentials for GCP authentication
- Docker Compose environment
