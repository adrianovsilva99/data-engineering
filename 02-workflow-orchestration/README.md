# 02 - Workflow Orchestration

This module is focused on workflow orchestration for data engineering pipelines.

The goal is to understand how orchestration tools help manage, schedule, monitor and execute data workflows in a structured and reliable way.

## Tools

This module will include experiments with two orchestration tools:

- Kestra
- Apache Airflow

## Current status

At the moment, the Kestra section has been completed as part of the Data Engineering Zoomcamp.

The Airflow section will be added later as an additional study to compare both tools and understand different approaches to workflow orchestration.

## Topics covered

- Workflow orchestration concepts
- ETL and ELT pipelines
- Scheduled executions
- Task dependencies
- Logs and monitoring
- Docker-based orchestration environment
- Google Cloud integration
- BigQuery and Google Cloud Storage workflows
